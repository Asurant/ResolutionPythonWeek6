# Resolution Python Week 6
I made a task manager tool using TKinter allowing you to keep track of all your tasks. You can add tasks at the top. You also are able to delete and complete tasks by clicking on the related button and then clicking on the items in the list in order to make your changes. You also are able to clear out all the completed tasks by pressing the related button.
